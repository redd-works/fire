INPUT_STEEL.PY DOCUMENTATION

Input_steel.py sets up the method required in order to read the steel_section.xlsx file, along with 2 other methods to calculate factored loads (loads) and section properties (sec_I). Different materials can be added to steel_section.xlsx.

input_steel.chooseSection()¶

This method will allow the user to select the sections he wishes to test, it does so by converting the excel file into a dictionary and then asking for user input. It will return the index of the chosen section in the dictionary.
